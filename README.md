## aiida-zeopp

AiiDA plugin for [Zeo++](http://www.zeoplusplus.org/)

# Installation

```shell
$ git clone https://github.com/ltalirz/aiida-zeopp aiida-zeopp
$ cd aiida-zeopp
$ pip install -e .  # also installs aiida, if missing (but not postgres)
$ reentry scan -r aiida  
```
